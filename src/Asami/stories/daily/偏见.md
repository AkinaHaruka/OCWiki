# 偏见

{{ v.asa }}的世界里却填满了电路、协议与代码。从基础的电子焊接、复杂的网络架构，到纯粹的软件开发与 Homelab 折腾，她都有所涉猎。也正因如此，她的社交软件里塞满了各类开源项目的交流群，或是氛围硬核的技术沙龙。

然而，在计算机与电子这类理工科底色浓厚的领域，{{ v.asa }}偶尔也会撞上一些挥之不去的偏见。

---

那是再平常不过的一天，{{ v.asa }}正划着手机，在某个技术群里看大家“吹水”。屏幕上，一位群友刚分享了自己的生活照。

- **{{ v.asa }}**：“姐妹好漂亮。”
- **希**：“我是男生哦。”

{{ v.asa }}隔着屏幕愣了半秒，随即失笑。在计算机圈子里待久了，这种“反差”倒也见怪不怪。她指尖飞舞，半开玩笑地回复道：

- **{{ v.asa }}**：“那你是男生的话，我是什么？会来月经的哥布林吗？”

这句自嘲瞬间像在平静的水面上投下了一枚深水炸弹，原本讨论代码的群友们纷纷冒泡。

- **波波**：“这种技术群里竟然会有女生？我是不太信的。”

虽然也有理性的群友指正“技术圈其实也有不少女性”，但质疑的声音依然此起彼伏。

{{ v.asa }}靠在椅子上，静静地注视着滚动跳跃的对话框。她本可以置之不理——毕竟她的能力无需性别背书，她更没有向陌生人自证身份的义务。

“但如果我现在站出来，”她心底泛起一丝波澜，“是不是能给那些‘女性不适合学理工’的陈旧观念，留下一道小小的裂痕呢？”

想到这里，{{ v.asa }}打开相册，翻出一张前几天保存的学工系统截图。在抹去了姓名和学号等隐私信息后，她将那张带有专业和性别标识的图片发到了群里。

屏幕那头陷入了短暂的死寂，随后是一行略显尴尬的文字。

- **波波**：“还……真有啊。”

**本节完**

## 作者的话

求知欲从不分性别，任何学科的大门都应为所有人敞开。

不可否认，目前理工领域中男性的占比更高，但这绝不应成为忽视女性存在的理由，更不该成为衡量能力的标尺。她们同样拥有卓越的天赋与毅力，在代码与电路的世界里，她们从未缺席。

性别不该是限制思想的枷锁。每个人都有权在知识的海洋里肆意探索，无论你是谁。

如果你也曾听过“女生不适合学理科”这类武断的论调，请务必相信：**决定你未来的绝非性别，而是你的智慧、勇气，以及永不熄灭的好奇心。**
