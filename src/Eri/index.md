# 浅见绘理

绘理 是我的第一个 OC，也是我第一次尝试写自己的 OC。

或者说，绘理其实就是我在虚拟世界的投影。她拥有和我相似的爱好，技能，与设定。

但不同的是，我并没有把她设定为一个 MTF，因为我希望她能够代替我在虚拟的世界里无忧无虑的生活下去。
